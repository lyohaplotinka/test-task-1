## Тестовое задание
#### Первый компонент: поиск с автоподсказкой
Автоподсказка срабатывает по таймауту при вводе. 
Файл `server/index.js` простейшим образом эмулирует работу сервера с 
тремя URL. Количество URL, к которым будет обращаться приложение, 
настраивается в .env (`MAX_SERVERS`). Количество результатов, которое
необходимо для остановки поиска указывается здесь же (`REQUIRED_RESULTS`).

#### Второй компонент: ввод номера по маске
Для введения номера по маске `+7 (XXX) XXX-XX-XX`. Не позволяет вводить
большее кол-во символов, не позволяет вводить не числовые символы.

#### Кнопки
Кнопка "Сохранить" записывает значение в localStorage с ключом `testTaskFormData`.
Кнопка "Очистить" удаляет значения из полей и записывает пустоту в localStorage.  
Также кнопка "Сохранить" отправляет значения на сервер в формате JSON 
методом POST.

При перезагрузке в форму подставляются значения из localStorage, 
если они есть и они валидны.

URL сервера настраивается в .env (`SERVER_URL`).

#### Локальная проверка
1. `git pull https://github.com/lyohaplotinka/test-task-1.git .`
2. `npm install`
3. `npm run start`
